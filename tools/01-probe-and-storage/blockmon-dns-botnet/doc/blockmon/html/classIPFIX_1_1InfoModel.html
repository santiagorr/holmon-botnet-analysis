static InfoModel& IPFIX::InfoModel::instance ( )
inlinestatic

InfoModel instance accessor.

InfoModel is a singleton; this class accessor ensures only one InfoModel exists at once.

At startup time, your application should initialize this instance using one of the three initialization methods, defaultV9(), defaultIPFIX(), or default5103().
