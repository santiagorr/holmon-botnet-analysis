Detailed Description

A libfc transcoder. Transcoders handle low-level encoding, decoding, and endianness issues. Each transcoder wraps an external buffer, owned by the client.

Transcoders are used extensively within libfc's internals (Exporter and Collector). They are only exposed to libfc clients through the SetReceiver interface. See the documentation for SetReceiver.receiveSet() for detailed information.
