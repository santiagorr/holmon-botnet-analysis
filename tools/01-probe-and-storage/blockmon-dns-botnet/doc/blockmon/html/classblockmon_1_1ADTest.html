const HistoMsg *  histo_msg,
bool  l_phase 
)
inline

Helper function used to update the windows hash table entry (window array) pointed to by the iterator passed as parameter. If in learning phase the function adds the current pdf to the end of the window array containing the list of window_elements included in the window of the current traffic feature. If in detection phase the function removes the first (oldest) element (window_element) from the array pointed to by the iterator, and add a new element containing the current histogram (pdf).

Parameters
w_itthe windows hash table iterator pointing to the windows hash table entry (<featiId, window_array> pair) of the required traffic feature
histo_msgthe current incoming HistoMsg
l_phasetrue if learning phase false if detection phase
